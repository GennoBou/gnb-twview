# gnb-twview

Twitchを効率的に視聴するために設計された、専用のデスクトップおよびモバイル向けブラウザです。

## 概要

`gnb-twview` は、Twitchのライブ配信を視聴することに特化した軽量ブラウザです。一般的なブラウザの余分な機能を削ぎ落とし、巡回オートモードなど、配信観賞を快適にするための機能を提供します。

### 主な機能 (v0.1.0 計画)
- **オートモード**: ライブ配信中のフォロー中配信者を、スマートキューシステムに基づいて一定時間ごとに自動で切り替えて巡回します。
- **スキップボタン**: キュー内の次の配信者へ即座に切り替えます。
- **ミニマルなUI**: 巡回コントロール、URLエリア、設定ボタンを配置した上部ナビゲーションバーと、配信画面のみのシンプルな二段重ね構成。
- **マルチウィンドウ統合**: 操作用のメインウィンドウ、カスタムCSSインジェクションで公式配信を描画しメインウィンドウの前面に固定されるサブウィンドウ（`TwitchWindow`）、および操作ラグや配信の再描画を防ぐため手前にポップアップ表示される独立した設定専用ウィンドウの3窓構成。
- **カスタムインジェクション (CSS/HTML/JS)**: Twitchウィンドウに対して、ユーザー独自のCSSスタイル、HTML要素、およびJavaScriptロジックを直接埋め込むことができる高度なカスタマイズ機能。入力時はリアルタイムで構文の文法エラーチェックが実行されます。
- **多言語GUI**: 英語および日本語に対応。初回起動時にシステム言語を自動検出し、設定画面からリアルタイムで表示言語を動的に切り替えることができます。

## 技術スタック
- **フレームワーク**: [Wails v3](https://v3.wails.io/)
- **フロントエンド**: Svelte v5 + Skeleton v4 + TailwindCSS (Skeletonの依存関係として使用)
- **バックエンド**: Go言語
- **対象プラットフォーム**: Windows (デスクトップ版), Android (タブレット/スマホ)
